# 修改日誌 (CHANGELOG)

## [2.0] - 2026-01-01

### 新增功能 (New Features)

#### 1. Hugging Face 數據源集成
- **完整支持 22 種加密貨幣**: AAVE, ADA, ALGO, ARB, ATOM, AVAX, BCH, BNB, BTC, DOGE, DOT, ETC, ETH, FIL, LINK, LTC, MATIC, NEAR, OP, SOL, UNI, XRP
- **多時間框架支持**: 15分鐘K線和1小時K線
- **自動數據下載**: 從 Hugging Face 數據集 (zongowo111/v2-crypto-ohlcv-data) 自動下載
- **自動緩存機制**: 首次下載後自動緩存到本地，無需重複下載

#### 2. GUI 改進

**數據加載界面**:
- 動態下拉菜單選擇幣種
- 動態下拉菜單選擇時間框架
- 一鍵從 Hugging Face 加載數據
- 實時加載進度提示

**數據預覽**:
- 表格顯示最新 20 筆K線數據
- K線圖表視覺化顯示
- 支持顯示最後 1000 根K線

#### 3. 完整的機器學習流程

**模型集成**:
- LightGBM 模型
- CatBoost 模型
- Ensemble 集成 (概率平均)

**特徵工程**:
- 基礎特徵: 收益率、高低差、開收差
- 移動平均線: SMA(5), SMA(10), SMA(20)
- RSI 指標: RSI(5), RSI(10), RSI(20)
- 波動性: 14 週期收益率標準差

**訓練流程**:
- 可配置的 Lookback 週期
- 可配置的訓練/測試分割比例
- 自動特徵標準化
- 模型持久化存儲

#### 4. 預測與導出

**預測功能**:
- 支持加載已訓練的模型
- 自動查找配套的 CatBoost 和 Scaler 文件
- 完整數據集預測
- 三種預測結果 (LightGBM, CatBoost, Ensemble)

**結果導出**:
- CSV 格式導出
- 包含時間戳和概率信息
- 易於進一步分析

### 技術改進 (Technical Improvements)

#### 多線程架構
- DataLoaderWorker: 非阻塞式數據加載
- TrainingWorker: 非阻塞式模型訓練
- UI 反應性提升

#### 錯誤處理
- 異常捕獲和詳細錯誤提示
- 網絡連接超時處理
- 數據驗證

#### 性能優化
- 智能緩存利用
- 向量化計算
- 內存高效的數據處理

### 文檔更新 (Documentation)

- **README.md**: 完整的功能說明和數據結構文檔
- **QUICKSTART.md**: 快速開始指南和常見問題
- **CHANGELOG.md**: 本修改日誌
- **requirements.txt**: 完整的依賴列表

### 已知局限 (Known Limitations)

1. **實時性**: 當前使用歷史數據進行訓練，非實時預測
2. **準確率**: 基礎 Ensemble 準確率約 51-54%，需結合其他指標
3. **單文件架構**: 當前為單一 Python 文件，大型項目可能需要模塊化
